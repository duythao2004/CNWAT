Git & GitHub

OWASP & Burp

Nắm các nhóm rủi ro Top 10, thao tác intercept/request để hiểu bề mặt tấn công.

Hướng dẫn sử dụng git cơ bản:

  1. Pull dự án từ GitHub trên máy hiện tại.
  2. Mở thư mục bằng VS Code → bật Live Server (nếu có).
  3. Tạo/ sửa trang trong mục bt2/ (giữ CSS dùng chung).
  4. Commit thay đổi, sau đó Push.
  5. Sang máy thứ hai và Pull để tiếp tục.
